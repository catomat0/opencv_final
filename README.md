외곽선 추출 이미지를 통한 배경화면 생성기

| 기술               | 설명                                    |
| ---------------- | ------------------------------------- |
| Python 3.x       | 전체 로직 구현 및 모듈 구성                      |
| OpenCV (cv2)     | 이미지 처리, 외곽선 추출(Canny), 색상 변환          |
| NumPy            | 이미지 배열 데이터 처리                         |
| PyQt5            | GUI 구성, 윈도우, 버튼, 레이아웃, 이미지 출력         |
| QPainter         | 배경 캔버스 렌더링, 실루엣 이미지 그리기               |
| QImage / QPixmap | OpenCV(Numpy) 이미지를 PyQt에서 렌더링하기 위한 변환 |
| Qt Mouse Events  | 마우스 드래그 이동, 선택 영역 처리                  |


project/
├── main.py                  # 프로그램 실행 진입점
├── main_window.py           # 메인 GUI 화면 및 캔버스 관리
├── image_editor_dialog.py   # 이미지 업로드, 외곽선 추출, 영역 선택
├── main_canvas.py           # 캔버스 이미지 배치, 드래그 이동
└── image_utils.py           # OpenCV -> QImage 변환 유틸리티


기능 흐름

프로그램은 아래 과정에 따라 배경화면을 생성한다.

1. 배경화면 사이즈 선택

2. 이미지 파일 업로드

3. Canny 외곽선 추출

4. 드래그 박스를 이용해 원하는 영역 선택

5. 선택된 실루엣 이미지를 메인 캔버스로 전송

6. 캔버스에서 이미지 드래그 이동

7. 배경색 변경 기능 활용

8. 최종 배경화면 이미지 저장
